# 소분류 재구성으로 끊긴 기술 — 연결(merge) 대상 목록

소분류명 변경/슬롯 재활용으로 키 `(분야, prefix, item_no)`가 바뀌면서
**같은 기술이 삭제+신설로 갈라진** 케이스를 모은다.
나중에 manual 매칭 CSV로 연결해 도입시기·연혁을 보존한다 (delete+new 방식은 신설일 리셋 부작용 때문에 폐기).

> 검토하며 발견 시 계속 추가. 형식: 분야/prefix · 전환 버전 · 옛 키 → 새 키 · 비고

## 신성장·원천기술 (tech)

### 7/라 — 바이오 화장품 소재 → 바이오 화학 (v200211)
- 기술: **「바이오 화장품 소재(원료) 개발 및 제조기술」** (이름·설명 완전 동일)
- 옛: `(7, 라, "")` 소분류 "바이오 화장품 소재", 2017 신설 → 2019-12-31 삭제
- 새: `(7, 라, "2)")` 소분류 "바이오 화학", 2020 신설 (current)
- 연결 시: 2017 도입 → 2020 소분류 이동(내용 동일), first_apply_date 2017 보존

### 9족 융복합소재 — v220215 통째 재편 (라·마·바·사 동시) — ⚠️ 상세 검토 필요
- 구리합금: 마(2020~21) → 라(2022~)
- 특수강: 사(2020~21) → 마(2022~)
- 타이타늄: 라(2017~21) → 사(2022~, "희소금속ㆍ소재" 산하)
- 몰리브덴: 바 → ("기능성 탄성ㆍ접착소재"로 슬롯 교체, 별개 가능성)
- 개별 기술 단위로 옛↔새 item_no 대응을 검토 후 확정

## ✅ 분야(sector) 번호 재정비 — 완료 (2026-06-09)
canonical 맵(`STRATEGIC_SECTOR_NO`/`NEWGROWTH_SECTOR_NO`)으로 개명 전후를 같은 번호로 병합:
- 신성장 1~14: 3=SW(옛3+12), 8=에너지(옛8+14), 14=방위산업. 비고 유령(16/18) 제거.
- 전략 1~8: 6=미래형 운송 및 이동수단(옛6+9), 8=인공지능. 비고 유령(8) 제거.
- 효과: 현행 활성 수 보존(전략81/신성장284), SW·에너지·미래형 기술 도입시기 연혁 복원, 가짜 폐지 제거.
- 남은 검토: **8(에너지)이 14·15로 쪼개진 건** — 탄소중립(13)으로 이동한 기술은 여전히 8에서 폐지+13 신설로 갈림. 기술 단위 검토 후 연결 여부 결정.

## (이력) 분야 단위 — 개명으로 번호가 갈라진 경우 [위에서 해결]
> 분야 번호 = `_norm_sector(분야명)` 최초 등장 순 누적. 개명 시 새 번호, 옛 번호는 공백.
> 합치면(canonical 매핑) 도입시기·연혁 보존 + 옛 분야 폐지행 오염 제거. 웹 표시는 이미 현행명만 노출.

- **3 → 12**: `차세대SW(소프트웨어) 및 보안`(2017) → `차세대소프트웨어(SW) 및 보안`(2019). 같은 분야.
  - 옛 3: 현행활성 0 / 현행폐지 9 (전부 2019 삭제) · 새 12: 현행활성 9. 도입시기 2017→2019로 끊김.
- **8 → 14 (+15 분리)**: `에너지신산업·환경`(2017~21) → `에너지ㆍ환경`(2022) + `탄소중립`(2022 신규 분리).
  - 옛 8: 현행활성 0 / 현행폐지 49 · 새 14: 활성 30 · 15: 활성 58. 에너지 기술 도입시기 끊김.
  - ⚠️ 8이 14와 15 **둘로 쪼개짐** — 어느 기술이 14로, 어느 게 15로 갔는지 기술 단위 검토 필요.
